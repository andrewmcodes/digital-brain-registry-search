# Digital Brain Registry Search Changelog

## [Initial Version] - 2026-07-09

- Search npm, RubyGems, GitHub, Homebrew, VS Code, and Obsidian community plugins from a single command.
- View generated `software`-note YAML frontmatter for any result.
- Copy frontmatter to the clipboard or create a note directly in an Obsidian vault.
- Preferences for vault path, software folder, and an optional GitHub token.
